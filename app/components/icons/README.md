# Icons Folder

Put static SVG icon components in this folder.

Example files:
- `Eye.vue`
- `EyeOff.vue`
- `cart/CartOutline.vue`

Usage:
- `<BaseIcon name="eye" />`
- `<BaseIcon name="eye-off" />`
- `<BaseIcon name="cart-outline" />`
- `<BaseIcon name="cart/cart-outline" />`

Notes:
- Use `currentColor` inside SVG paths/fills when possible.
- Keep icon components as plain `<svg>` roots for best compatibility.
